# Translate:Tool
for Nertivia

## Requirements
- Python
- Node.js

## Install
1. Put your bot token on the last line, replacing the placeholder.
2. `npm install`.
3. Edit `main.py`, and put the directory where it is `bot.js` on the placeholder (remember using double `\` if you are using Windows).
4. `python main.py`.
